# Plugin Reloader

Enable or disable plugins from within the editor main screen.

[**Download**](https://github.com/kenyoni-software/godot-addons/releases)

## Compatibility

| Godot | Version  |
|-------|----------|
| 4.4   | >= 1.0.0 |
| 4.3   | >= 1.0.0 |
| 4.2   | >= 1.0.0 |
